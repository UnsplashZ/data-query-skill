## 来源结构口径
不要只写：`BD1-KOL 12.03万 / 占52.29%`。

应写为：

```text
BD1-KOL GMV xx万 / 退费 xx万 / 退费率 x.xx%
```

原因：按退费额排序只能说明影响规模，不能说明来源质量；必须同时看分母规模和来源内退费率，避免把大体量来源误判为唯一问题，也避免过度解读小体量高退费率来源。

## 实现提示
在 pandas 拆解表里保留：
- `分组GMV`
- `退费金额`
- `退费金额占比`
- `分组内退费率 = 退费金额 / 分组GMV`

报告正文可用 helper：

```python
def source_line_with_rate(diag, focus, dim='渠道归属', n=3):
    sub = diag[(diag['重点cohort'] == focus) & (diag['维度'] == dim)].sort_values('退费金额', ascending=False).head(n)
    return '；'.join(
        f"{r['分组']} GMV {money_w(r['分组GMV'])} / 退费 {money_w(r['退费金额'])} / 退费率 {pct(r['分组内退费率'])}"
        for _, r in sub.iterrows()
    )
```

来源图可使用 `draw_source_barh()`，标签同时显示 `退费额｜率｜GMV`。
