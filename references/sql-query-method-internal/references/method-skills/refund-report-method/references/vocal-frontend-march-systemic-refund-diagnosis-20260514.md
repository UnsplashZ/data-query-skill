# 声乐前端 3月系统性退费诊断补充（2026-05-14）

## 适用场景

当用户追问“3月 M0/M1 都高是否不合理”“是否不是某个渠道问题”“需要看退款原因里的猫腻”时，不能只用绝对退费额或单一渠道归因，要把渠道退费率、历史月均原因增量、原始退款文本、支付到退费时效放在一起判断。

## 已验证口径

- 范围：声乐 + 前端 + 课程 + 全款/尾款。
- GMV：按支付月归属，作为 cohort 退费率分母。
- M0/M1：按 `refund_time` 相对 `pay_time` 所在月计算。
- 渠道比较：必须用 `分组内退费率 = 退费金额 / 分组GMV`；绝对退费额只用于判断影响规模和处理优先级。
- 3月/4月重点情况必须同时展示当月 GMV 分母，避免用户无法校验退费率。

## 关键发现

### 1. 3月不是单一 BD1-KOL 事故

BD1-KOL 3月 M0 退费率 4.91%，历史 M0 大多在 0.39%~1.41%，确实异常；但 3月 M0 还有其他高退费率渠道：

- 工作室1-中小：5.94%
- 直播-自然流：4.88%
- BD2-图书：4.68%
- BD1-书课包：4.54%

结论应写成：3月不是单一渠道事故，而是全局风险抬升；BD1-KOL 因为 GMV 大且退费率高，放大整体退费额。

3月 M1 也不应只盯 BD1-KOL：

- 工作室1-信息流：3.17%
- 析出-企微&公众号：2.69%
- BD1-KOL：2.49%

### 2. 3月 M0 较历史月均的原因增量

相比 2025-10~2026-02 的 M0 历史月均，3月 M0 增量来自：

- 健康/家庭：+7.98万，占增量 53.45%
- 其他长尾/未填：+3.38万，占 22.61%
- 时间冲突：+2.06万，占 13.77%
- 产品与服务匹配：+1.49万，占 9.96%
- 支付/价格机制：+0.03万，占 0.20%

结论：异常不在支付错误，而在成交前风险筛查、家庭/时间/健康约束确认、承诺与交付预期、D4-D15 承接体验。

### 3. 原始退款原因和时效信号

3月 M0 原始原因 Top：

- 家人原因，不让学习：5.82万，平均支付到退费 8.5 天
- 身体原因：4.56万，平均 12.7 天
- 时间原因：2.49万，平均 11.1 天
- 错误付款：1.76万，平均 3.9 天
- 重复付款：0.91万，平均 3.6 天
- 无法满足学员特殊要求：0.67万，平均 13.0 天
- 课程原因：0.62万，平均 5.7 天

前三项不是 D0-D1 即时误购，而是支付后首周到半个月集中暴露。

### 4. 原始文本中应保留的业务信号

报告正文可引用这些主题，不要只写结构化原因：

- 合同太宽泛 / 内容不明确
- D6 解散群且未建新班级群导致不信任
- 着急开课但老师未分配，甚至触发 12315 投诉
- 课程 / 赠品不满意
- 家人生病、急需用钱、贷款等家庭财务约束

这些信号指向成交承诺、合同认知、分班开课、辅导老师承接、家庭财务约束共同导致的系统性风险。

## 报告写法建议

报告应包含：

1. 问题：3月支付 cohort 当月退费异常高，且多个渠道退费率同时抬升，不是 4月追交接率能解释。
2. 原因：成交质量、预期确认、开课承接体验组合问题。
3. 证据：渠道退费率、历史原因增量、原始退款文本、D4-D15 时效。
4. 建议：复盘 3月活动/政策/直播话术/合同/赠品/开课排班；对“其他长尾/未填”做文本归类；追 D4-D15 订单的入群、分班、老师分配、开课、投诉节点。
